# Petites Annonces

Site web de petites annonces en ligne développé en PHP POO (Programmation Orientée Objet) avec une architecture MVC.
Reprise d'un ancien projet en PHP Procédural.

## Fonctionnalités

- Dépôt et consultation d'annonces (vente, immobilier, véhicules, articles déco...)
- Inscription et connexion des membres
- Système de commentaires et de notes
- Back-office d'administration
- Filtres et recherche d'annonces
- Statistiques

## Technologies

- PHP 8.2 (POO / MVC)
- MySQL / PDO
- Bootstrap 5
- MAMP (environnement local)
